The Morning Club

Financial news aggregation platform with AI sentiment analysis (Under Development - Private Beta)

Overview

The Morning Club is a comprehensive platform designed to streamline financial news consumption. By leveraging advanced AI models, it automatically aggregates, summarizes, and analyzes the sentiment of daily business news, allowing users to stay informed without spending hours reading through articles.

The platform specifically focuses on key tech companies like NVIDIA and AMD, providing daily metric reports and actionable insights. It also features an educational community forum where users can discuss trends and share knowledge.

Key Features

AI Summarization

Automatically condenses long financial articles into quick, readable summaries.

Sentiment Analysis

Classifies news sentiment to gauge market reactions instantly.

Daily Reports

Automated generation of key business metrics for targeted tech companies.

TTS Integration

Text-to-Speech technologies to make consuming articles faster and more accessible.

Tech Stack

React & Tailwind
Frontend framework and styling for a responsive, modern UI.
Supabase
Backend-as-a-Service for database, authentication, and real-time subscriptions.
n8n
Workflow automation tool to orchestrate data fetching and AI processing.
AI Models
Integration of the best LLMs for natural language processing tasks.

Astralis (AI Assistant)

Multi-model AI assistant with voice synthesis and automation (Discontinued)

Overview

Astralis is an intelligent conversational assistant built to automate daily tasks and provide seamless interactions. By integrating multiple AI models through OpenRouter and n8n, it handles complex queries, manages calendars and emails, and scrapes web data efficiently.

A standout feature is its voice synthesis capability powered by ElevenLabs, giving the assistant a natural and engaging voice. It also features a call capability, allowing it to call telephones and speak to make reservations or ask for details.

Note: This project has been discontinued as there are now better options available on the market. It remains in the portfolio as a demonstration of complex API integration and automation.

Key Features

Voice Synthesis

Natural-sounding voice responses using ElevenLabs API.

Calendar & Email

Automated scheduling, event tracking, and email handling.

Phone Call Feature

Can call telephones and speak to make reservations or ask for details.

Web Scraping

Intelligently extracts information from websites to answer queries.

Tech Stack

n8n
Core automation engine connecting various APIs and services.
OpenRouter
Unified interface to access multiple LLMs for different tasks.
ElevenLabs
High-quality text-to-speech generation.

Social Media Audit Bot

Automated social media account analyzer and report generator

Overview

This automated agent provides comprehensive reviews of social media accounts, systematically analyzing titles, portraits, content quality, and engagement metrics (comments, likes).
